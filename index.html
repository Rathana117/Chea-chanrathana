Recent Projects

Project E-Commercial

I developed a full-featured e-commerce web application that allows users to browse products, add items to a shopping cart, and securely check out. The platform includes features such as user authentication, product search and filtering, order tracking, and an admin dashboard for inventory management. The front end was built with React and Tailwind CSS, while the back end uses Node.js, Express, and MongoDB. Payment integration was handled using Stripe API. This project demonstrates my ability to build scalable, user-friendly web applications with real-world functionality.

Project Crypto

I built a cryptocurrency dashboard that provides real-time market data, historical charts, and portfolio tracking. The application fetches live data from the CoinGecko API and displays interactive charts using Chart.js. Users can search for specific coins, view detailed information, and simulate buying and selling to manage a virtual crypto portfolio. The front end was developed with React and Tailwind CSS, while the backend (optional for auth and saved portfolios) uses Node.js and MongoDB. This project highlights my ability to work with third-party APIs, data visualization, and responsive UI design.

Project WEB3

I created a Web3 decentralized application (dApp) that allows users to connect their crypto wallets and mint custom NFTs on the Ethereum blockchain. The front end was built with React and uses Web3.js and Ethers.js to interact with smart contracts deployed via Solidity on the Ethereum testnet. Users can sign transactions, view their owned NFTs, and explore a marketplace section. The smart contracts were tested using Hardhat and deployed through Alchemy. This project demonstrates my understanding of blockchain fundamentals, smart contract development, and integrating decentralized technologies into modern web apps.
